Unrefereed preprint.
- [The Burau representation of the braid group is faithful for n = 4 (Bharathram, Birman, Brendle)](https://arxiv.org/abs/2607.05283) — resolves the last open case of a decades-old braid-group / low-dimensional-topology problem; corollary: the Jones representation of B₄ is faithful too. Unrefereed preprint.
- [An exponential improvement for Ramsey lower bounds (Ma, Shen, Xie)](https://arxiv.org/abs/2507.12926) — first exponential gain over Erdős' classical 1947 probabilistic lower bound for r(ℓ, Cℓ); anchors the Ramsey trend.
