## Worth studying

- [Kakeya set conjecture in three dimensions — resolved (Wang, Zahl)](https://arxiv.org/abs/2502.17655) — every Kakeya set in ℝ³ has Minkowski and Hausdorff dimension 3, via a volume estimate for unions of δ-tubes not concentrated in a convex set (geometric measure theory / harmonic analysis); the landmark behind the 2026 Clay Research Award.
- [The Algebraic Montgomery–Yang Problem (Jo, Park, Park)](https://arxiv.org/abs/2607.06686) — resolves Kollár's conjecture: a rational homology projective plane with quotient singularities and simply-connected smooth locus has at most three singular points (algebraic geometry ↔ 4-manifold topology). Unrefereed preprint.
- [The Burau representation of the braid group is faithful for n = 4 (Bharathram, Birman, Brendle)](https://arxiv.org/abs/2607.05283) — resolves the last open case of a decades-old braid-group / low-dimensional-topology problem; corollary: the Jones representation of B₄ is faithful too. Unrefereed preprint.
- [An exponential improvement for Ramsey lower bounds (Ma, Shen, Xie)](https://arxiv.org/abs/2507.12926) — first exponential gain over Erdős' classical 1947 probabilistic lower bound for r(ℓ, Cℓ); anchors the Ramsey trend.
- [GWTC-5.0 — LIGO–Virgo–KAGRA gravitational-wave catalog](https://www.ligo.org/news.php) — updated GW transient catalog; the current compact-binary-merger census.
